# Threat Model

Actors:
- Opportunistic scammers using cloned or deepfaked celebrity/brand media.
- Organized groups creating large-scale sybil networks to amplify scams.
- Advanced adversaries producing high-quality deepfakes to bypass detectors.

Attacker capabilities:
- Create synthetic video/audio and host on platforms.
- Register domains and smart contracts to lure victims.
- Coordinate repost networks and bot amplification.

Defenses:
- Multimodal detection, human-in-the-loop review, takedown workflows, and on-chain forensic tooling.
